## Installation Guide
Create a python environment for the specific multimodal large language model, activate it and install the required packages. 

## OneLLM

```bash
conda create -n onellm python=3.9 
conda activate onellm
cd OneLLM

pip install -r requirements.txt
```

## ChatBridge

```bash
conda create -n chatbridge python=3.9 
conda activate chatbridge
cd ChatBridge

pip install -r requirements.txt
```

## VideoLLaMA 2
```bash
conda create -n videollama2 python=3.9 
conda activate videollama2
cd VideoLLaMA2

pip install -r requirements.txt
```
